# KaviosPix Frontend

A modern image gallery frontend where users can authenticate with Google, create albums, upload images, comment on images, and manage their profile. Built with **React**, **TypeScript**, and **Vite**.

---

## Demo Link

[Live Demo](https://your-frontend-url.vercel.app)

## Quick Start

```bash
git clone https://github.com/<username>/kaviospix-frontend.git
cd kaviospix-frontend
npm install
npm run dev
```

Create a `.env` file.

```env
VITE_API_URL=http://localhost:5000/api
```

## Features

### Authentication

- Sign in using Google OAuth
- Persistent authentication using protected APIs
- Automatic session restoration

### Album Management

- Create, update, and delete albums
- View all albums
- Ownership-based album actions

### Image Management

- Upload images
- View album images
- Update and delete uploaded images
- Ownership-based image management

### Comments

- Add comments on images
- Edit and delete own comments

### Profile

- View profile information
- Update additional profile details
- Secure logout

### State Management

- Context API for authentication
- Redux Toolkit for album state management
